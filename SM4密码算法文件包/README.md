### 命令使用方法

```
./sm4 input key mode flag
 - input：为明文或者密文
 - key: 为密钥
 - mode: 为5中工作模式: ECB-0/CBC-1/OFB-2/CFB-3/CTR-4
 - flag: 为加密解密的标记，0-解密/1-加密，如果input为明文flag要设置为1，密文要设置为0.
 
 *.example：
  - 加密：./sm4 12121212121212121212121212121212 12121212121212121212121212121212 0 1
  - 解密：./sm4 773fec4633e4215bac032813f3f9dd68 12121212121212121212121212121212 0 0
 
```
